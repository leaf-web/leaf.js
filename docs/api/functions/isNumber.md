# leaf.isNumber

Determines if a reference is a `number`.

----------------------------------------------------------------------

## Usage

leaf.isNumber(value);

### Params

| Param           | Type          | Details                          |
| --------------- | ------------- | -------------------------------- |
| value           | `*`           | The reference to check.          |

### Returns

`True` if `value` is a `number`.

----------------------------------------------------------------------

## Example

	<html>	
		<body>
			<script src="scripts/leaf.min.js"></script>
			<script>	
				console.log(leaf.isNumber(1)); //returns true
			</script>
		</body>
	</html>
