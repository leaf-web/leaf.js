# leaf.isBoolean

Determines if a reference is a `boolean`.

----------------------------------------------------------------------

## Usage

leaf.isBoolean(value);

### Params

| Param           | Type          | Details                          |
| --------------- | ------------- | -------------------------------- |
| value           | `*`           | The reference to check.          |

### Returns

`True` if `value` is a `boolean`.

----------------------------------------------------------------------

## Example

	<html>
		<body>
			<script src="scripts/leaf.min.js"></script>
			<script>
				console.log(leaf.isBoolean(true)); //returns true
			</script>
		</body>
	</html>
