# leaf.isNull

Determines if a reference is `null`.

----------------------------------------------------------------------

## Usage

leaf.isNull(value);

### Params

| Param           | Type          | Details                          |
| --------------- | ------------- | -------------------------------- |
| value           | `*`           | The reference to check.          |

### Returns

`True` if `value` is `null`.

----------------------------------------------------------------------

## Example

	<html>
		<body>
			<script src="scripts/leaf.min.js"></script>
			<script>
				console.log(leaf.isNull('John Doe')); //returns false
			</script>
		</body>
