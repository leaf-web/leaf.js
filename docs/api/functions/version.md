# leaf.version

Returns the current version of leaf.js.

----------------------------------------------------------------------

## Usage

leaf.version

### Returns

`string` The version number.

----------------------------------------------------------------------

## Example

	<html>
		<body>
			<script src="scripts/leaf.min.js"></script>
			<script>
				console.log(leaf.version);
			</script>
		</body>
	</html>
